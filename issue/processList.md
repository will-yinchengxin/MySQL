## 查看 MySQL 当前的活跃进城信息

```sql
mysql> select * from information_schema.PROCESSLIST;
+----------+------+--------------------+----------+---------+-------+--------------+-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| ID       | USER | HOST               | DB       | COMMAND | TIME  | STATE        | INFO                                                                                                                                                                                                      |
+----------+------+--------------------+----------+---------+-------+--------------+-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| 24691524 | root | 172.16.27.40:33598 | ngvsm    | Sleep   |     0 |              | NULL                                                                                                                                                                                                      |
| 24691526 | root | 172.16.27.40:33594 | ngvsm    | Sleep   |     0 |              | NULL                                                                                                                                                                                                      |
| 24691521 | root | 172.16.27.40:33588 | ngvsm    | Sleep   |     0 |              | NULL                                                                                                                                                                                                      |
| 24681035 | root | 172.16.27.50:47006 | vrbm     | Sleep   |  5333 |              | NULL      
+----------+------+--------------------+----------+---------+-------+--------------+-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+          
````
执行这条命令时，MySQL 会返回当前所有活跃的进程列表，包括每个进程的详细信息，如进程 ID、用户、主机、数据库、执行的 SQL 语句等。这对于监控和调试数据库性能非常有用，因为它可以帮助你了解哪些进程正在运行，它们正在执行什么操作，以及它们已经运行了多长时间。
